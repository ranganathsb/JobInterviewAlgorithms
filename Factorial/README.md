# Factorial
**Problem Definition:** 
Write a function that takes a number as an argument and return its factorial value.

---
**Examples:**
* `Factorial(5) = 5 * 4 * 3 * 2 * 1 = 120`
* `Factorial(3) = 3 * 2 * 1 = 6`
* `Factorial(1) = 1`;

---
**Notes:**
* Common ways of solving this problem are through Iteration or Recursion.
